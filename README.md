# shell-productivity

My zsh/bash productivity aliases / tools / scripts

## Installation

### Step 1

```bash
cd ~
git clone https://github.com/selimaj-dev/shell-productivity.git
```

### Step 2 (zsh)

Add this into your `.zshrc`

```bash
source ~/shell-productivity/rc.zsh
```

## Usage

`ga` - **git add -A**
`gp` - **git pull**
`gc <message>` - **git commit -am <message>**
`postpr` - **Done after a pull request is merged, switches to master and removes the pr branch and pulls**
